A simple plugin (widget) to display Ziki informations

== Description ==

This plugin aims at displaying the Ziki profile ("http://www.ziki.com") in a sidebar widget.
You can display:
- avatar
- ziki link
- Ziki profile name
- Real name
- Baseline
- Profile links (technorati, facebook, linkedin...)
- Fans and friends number
- Feedburner and technorati stats
- Display last Twitter baseline

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `zikiplugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Actve the widget in the 'Presentation/Widgets' menu
